# /bardswap

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/bardswap [melee]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Toggles swapping instruments / weapons on and off
<!--cmd-desc-end-->

## Options

| Option | Description |
|--------|-------------|
| `(no option)` | Toggle swapping on and off. |
| `melee` | Toggle using "Melee swap" delay. |
